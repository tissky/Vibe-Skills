# Reference Archive

`references/archive/` 保存已经退出 active stable path 的历史 ledger、snapshot 或 changelog volumes。

这些文件仍然可追溯，但默认不再承担当前 release governance、runtime contract 或 operator navigation 的 live truth。

## Start Here

- [`changelog/README.md`](./changelog/README.md): 历史 changelog volumes
- [`proof-bundles/README.md`](./proof-bundles/README.md): 历史 proof bundles

## Rules

- 当前稳定路径仍优先保留在原位，例如 `references/changelog.md`。
- archive volume 用来承接历史长尾，不把 live stable path 改成漂移路径。
