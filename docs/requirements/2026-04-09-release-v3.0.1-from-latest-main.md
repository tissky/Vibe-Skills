# Release v3.0.1 From Latest Main

## Goal

Publish a new governed release from the latest upstream `main` for `foryourhealth111-pixel/Vibe-Skills`.

## Deliverable

- verified release branch `release/v3.0.1`
- updated governed release surfaces for `v3.0.1`
- pushed Git tag `v3.0.1`
- published GitHub release if the authenticated CLI path allows it

## Constraints

- baseline must be upstream `origin/main` as fetched on 2026-04-09
- work must happen in an isolated workspace, not the dirty local checkout
- release claims require verification evidence
- repository-native release tooling should be preferred over ad hoc edits

## Acceptance Criteria

- `config/version-governance.json`, changelog, release ledger, release docs, and dist manifests are aligned to `v3.0.1`
- release-critical installer regressions are fixed
- targeted release/install verification passes on the release branch
- `v3.0.1` tag is created from the verified release commit
- GitHub release is published or an exact blocker is recorded

## Product Acceptance Criteria

- release notes truthfully summarize changes since `v3.0.0`
- nested compatibility runtime behavior remains materially correct after install
- release metadata does not regress host-visible wrapper or runtime packaging projections

## Manual Spot Checks

- inspect `scripts/governance/release-cut.ps1 -Preview`
- inspect release diff before publication
- confirm tag and GitHub release state after publish

## Non-Goals

- merging unrelated local work
- broad non-release refactors
- claiming full publication if GitHub publish steps fail
